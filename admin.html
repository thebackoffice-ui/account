1. Connect Google Sheets

All data saves to a Google Sheet via Apps Script. One-time setup — takes about 5 minutes.

Create a new Google Sheet (name it anything, e.g. "The Back Office")
In the sheet go to Extensions → Apps Script
Delete existing code, paste in the Apps Script below (copy button top-right), and save
Click Deploy → New deployment → Web app · Set "Who has access" to Anyone · Deploy · copy the URL
In Apps Script go to Project Settings → Script Properties · Add property TOKEN_SECRET with a long random value (e.g. 40 random characters) — this secures admin tokens
Paste that URL into the field below and click Save URL — manager.html reads it automatically from the same browser storage

2. Connect Pay Reports

The Rep Pay tab in manager.html reads pay data from your Google Sheet's Week Index tab. Paste the CSV URL for that tab below — managers will see their team's pay reports automatically once this is saved.

In your pay reports Sheet, open the Week Index tab
Copy the sheet ID from the URL bar — it's the long string between /d/ and /edit
Build the URL: https://docs.google.com/spreadsheets/d/SHEET_ID/gviz/tq?tqx=out:csv&sheet=Week%20Index
Paste it below and click Save URL

The admin page also runs a fallback check on load as a backup.
